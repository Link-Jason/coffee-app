# Coffee-App (Midnight Rust Edition)

A precision-engineered brewing calculator designed for consistency. This tool provides real-time calculations for coffee-to-water ratios, wrapped in a high-contrast, "Midnight Rust" interface.

## Design Philosophy
* **Precision:** Distraction-free interface focused on the essential metrics.
* **Midnight Rust Aesthetic:** High-contrast, dark-mode-first styling using deep charcoal (`#0F0F0F`) and artisan rust (`#A65B3C`) accents.
* **Editorial Layout:** Professional, left-aligned typography for a "digital instrument" feel.

## Tech Stack
* **Framework:** React Native + Expo
* **Language:** TypeScript
* **Deployment:** Vercel

## Quick Start

```bash
# Clone the repository
git clone [https://github.com/Link-Jason/coffee-app.git](https://github.com/Link-Jason/coffee-app.git)

# Navigate to project
cd coffee-app

# Install dependencies
npm install

# Run locally
npx expo start
